# Sales Performance Analytics Dashboard 📊

## Project Overview
This project showcases an end-to-end data analysis workflow using Microsoft Excel. It transforms raw transactional data into an interactive, stakeholder-facing dashboard, accompanied by actionable business insights.

## Key Features & Workflow
The project was executed in five key phases:

- **✅ Phase 1: Data Preparation**
  - Loaded and formatted a raw dataset of **9,994 transaction records** into a clean, structured Excel table for robust analysis.

- **✅ Phase 2: Data Modeling**
  - Built 5 comprehensive Pivot Tables to analyze key business dimensions:
    - **Region**
    - **Category**
    - **Sub-Category**
    - **Monthly Trend**
    - **Segment**

- **✅ Phase 3: Data Visualization**
  - Developed 5 insightful visualizations including dynamic Bar Charts and a Line Trend Chart to highlight performance and seasonal trends.

- **✅ Phase 4: Interactivity**
  - Integrated 3 interactive Slicers connected across all 5 pivot tables, allowing seamless and intuitive filtering of the dashboard metrics.

- **✅ Phase 5: Business Insights & Recommendations**
  - Formulated a **Business Recommendation Brief** detailing critical findings (e.g., identifying *Technology* as the top-performing category, *West* as the most profitable region, and flagging *Tables, Bookcases & Supplies* for negative profitability) and actionable recommendations for stakeholders.

1. Download or clone this repository.
2. Open `Sample - Superstore.xlsx` in Microsoft Excel.
3. Use the interactive slicers on the Dashboard sheet to filter data dynamically.
4. Review the Business Recommendation Brief sheet for strategic insights.
